프로젝트 개요

파이썬의 기초와 웹 기초를 연결시킬 수 있는 좋은 프로젝트를 생각하던 중, 간단하지만 재미있는 키오스크를 만들면 좋을 것 같다는 생각이 들었다.

개발 계획

구현 요소

1. 만약 사람이 없으면 입장 가능

2. 입장시 인원수대로 2인, 3~4인, 5인 이상으로 나누어 대기

3. 사람들의 태이블이 자동 배정되며 입장 가능

4. 관리자 페이지 제작으로 사용자들의 현재 테이블 사용량 및 대기 인원 확인 가능

개발 환경

python

streamlit -> python 기반 웹 프레임워크
